## Dominance MCP  APPS

This is a sample application to demonstrate the use of MCP in a Spring Boot application.

## Prerequisites

- Java 17
- Maven 3.8.4
- Spring Boot 2.7.5
- MCP 1.0.0

## Setup

- Clone the repository
- Run `mvn clean install`
- Run `mvn spring-boot:run`

## Usage

- The application will start on port 8005
- The MCP tools will be available at `/mcp/sse`

## MCP Tools

- authenticate_user
- get_current_user_info
- refresh_token
